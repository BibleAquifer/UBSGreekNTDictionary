# _UBS Dictionary of the Greek New Testament_ Aquifer Resource

This repository (`UBSGreekNTDictionary`) contains Aquifer resource data with resource-level metadata, article-level metadata, and content files in JSON and Markdown formats.

## Structure

The repository is organized by language codes, with each language folder containing subfolders for JSON and Markdown content.
